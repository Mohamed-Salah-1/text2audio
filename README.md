# Text2Audio

A professional neural text-to-speech studio powered by free, high-quality TTS engines. Built and maintained by Artiforia-Hub.

## Features
- Natural-sounding voices (Edge TTS, Google TTS, Coqui TTS)
- Professional voice controls (pitch, speed, style)
- Modern Streamlit UI
- No paid APIs required

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
